Getting Started

Upload Your Data

Start by uploading your CSV or Excel file through the Analyze page. The system supports files up to 50MB.

2

Ask Questions

Use natural language to ask questions about your data. For example: "Show me the average sales by region" or "Find correlations between variables".

3

Get Results & Code

Receive instant analysis results along with the generated Python code that you can review, modify, or reuse.

File Upload Guide

Supported Formats

  • • CSV files (.csv)
  • • Excel files (.xlsx, .xls)

Best Practices

  • • Ensure your data has headers in the first row
  • • Remove any empty rows or columns
  • • Use consistent date formats
  • • Keep file size under 50MB for optimal performance

Data Preprocessing

The system automatically:

  • • Detects and converts date columns
  • • Handles missing values appropriately
  • • Converts numeric strings to numbers
  • • Standardizes text encoding

Troubleshooting

File Upload Issues

Ensure your file is in supported format and under 50MB. Check for special characters in column names.

Query Not Working

Try rephrasing your question. Be specific about column names and the type of analysis you want.

Slow Performance

Large datasets may take longer to process. Consider sampling your data or using more specific queries.

API Key Issues

Ensure your Groq API key is valid and has sufficient credits. Check the Settings page to update your API key.

Frequently Asked Questions

Is my data secure?

Yes, all data processing happens on your local machine or server. Your data is never stored on external servers unless you explicitly choose to export it.

What AI model does this use?

We use Groq's Llama models, specifically optimized for fast inference and code generation tasks.

Can I use my own Python code?

Yes! You can modify the generated code directly and run it again, or write your own custom analysis code.

Is there a limit to queries?

The limit depends on your Groq API key's rate limits. Most free tiers provide sufficient queries for regular use.

Can I export my analyses?

Yes, you can export results as CSV, Excel, or PDF, and the generated code as Python files.
